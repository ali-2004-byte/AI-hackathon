# Prompt Engineer (v1.0.1)

Expert system for transforming raw intent into state-of-the-art structured prompts using 12 elite industry frameworks.

## Core Capabilities

### 1. Automatic Framework Selection
The skill automatically maps your task to the most effective framework:
- **Programming/Technical**: RODES (Role, Objective, Details, Examples, Steps)
- **Complex Logic**: Chain of Thought + RTF (Role, Task, Format)
- **Information Density**: Chain of Density (Recursive simplification)
- **Problem Solving**: STAR (Situation, Task, Action, Result)
- **Business/Project**: RISEN (Role, Instructions, Steps, End Goal, Narrowing)

### 2. Zero-Config Optimization
Simply provide a rough concept or prompt, and it will output a production-ready instruction block.

## How to Use

Invoke this skill by providing a rough prompt you want to optimize.

**Examples:**
- `prompt-engineer create a prompt for building a distributed cache`
- `prompt-engineer optimize: analyze this log for errors using complex logic`

## Success Indicators
- **Clarity**: Removes ambiguity from instructions.
- **Constraints**: Forcefully defines output formats (Markdown, JSON, etc.).
- **Expertise**: Assigns highly specific personas (e.g., "Senior L8 Software Architect").
