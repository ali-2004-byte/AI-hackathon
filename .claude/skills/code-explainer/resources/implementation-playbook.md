# Code Explanation and Analysis Implementation Playbook

Detailed patterns and checklists for code explanation.

## 1. Code Complexity Analysis
Identify async/await, decorators, generators, and comprehensions to gauge difficulty.

## 2. Visual Explanations
Use Mermaid `flowchart TD` or `classDiagram` to visualize logic.

## 3. Step-by-Step Breakdown
1. High-level summary of purpose.
2. Step-by-step logic flow.
3. Deep-dive into complex concepts (e.g., closures, recursion).

## 4. Common Pitfalls
Identify "Bare except", "Magic strings", or "Global variables" and suggest refactors.
